Steps to install, initialize and run eslint:
- After creating an index.js file, run the command "npm init".
- This will add package.json file to the project.
- Now run "npm install eslint --save-dev". This will install eslint package to the project.
- Now initialize eslint using the command "npx eslint --init".
- This will ask a bunch of questions and choose according to your type of project.
- Now use eslint using the command "npx eslint <your file>.js".
